# SIO-K9 v0.5.0 — Backup automático no Google Drive

## Novidades
- Configuração da URL /exec e token diretamente na aba Backup.
- Backup automático após salvar, editar, excluir ou restaurar dados.
- Botão Fazer backup agora.
- Teste de conexão com o Apps Script.
- Restauração do backup mais recente do Google Drive.
- Mantidos backup JSON local e exportação de relatório em PDF.

## Segurança
O token é salvo somente no armazenamento local do navegador. Não publique o token no GitHub nem envie em capturas de tela.
